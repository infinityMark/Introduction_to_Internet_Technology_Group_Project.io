SEHH2041 Applied Computing

Upon completion of this subject, students will be able to

  1. outline information technology and its application in different disciplines
  2. describe the functions and features of computer hardware and software components
  3. explain the principles of computer networks and data communications
  4. describe the basic structure of a database system and its application
  5. discuss the basic functions of a computer operating system
  6. develop end-user computing and problem-solving skills for use in real-life situation

Continuous Assessment: 40% + Examination: 60% = 100%


SEHH2042 Computer Programming

Upon completion of this subject, students will be able to

  1. develop computer programs in one or more high level language programming environment
  2. design and develop structured and documented computer programs
  3. explain the fundamentals of object-oriented programming and apply it in computer program development
  4. integrate the computer programming techniques to solve practical problems

Continuous Assessment: 50% + Examination: 50% = 100%


SEHH1069 Calculus and Linear Algebra

Upon completion of this subject, students will be able to

  1. describe the concepts of limit and continuity of functions, derivatives of functions, indefinite and definite integrals
  2. evaluate limits, derivatives and integrals of functions in a single variable
  3. implement techniques of solving systems of linear equations
  4. perform basic operations of matrix algebra and determinants
  5. apply methods of calculus and linear algebra to solve a range of practical problems

Continuous Assessment: 40% + Examination: 60% = 100%


SEHH2240 Database System

Upon completion of this subject, students will be able to

  1. describe the basic concepts of database systems
  2. describe the architecture and functioning of database management systems, as well as to be able to apply the associated tools and techniques
  3. apply the principles and practices of good database design and analysis
  4. demonstrate an understanding of the direction of database technology and their implication so as to manage and plan database system developments
  5. apply database technologies for lifelong learning, e.g., web databases

Continuous Assessment: 50% + Examination: 50% = 100%
